1. Roles & Permissions
🧑‍💼 Super Admin (Platform Owner)

Can:

Create / approve / suspend business accounts

Manage all users (customers + business admins)

Define global reward rules and templates

See platform-wide analytics

Moderate content (photos, quizzes, submissions)

Configure system settings (plans, billing, feature flags)

🏪 Business Admin (Per Business)

Can:

Manage their own business profile

Create & manage listings (locations, missions, activities)

Create & manage rewards for their customers

View & manage their users/participants

Approve/verify submissions (photos, receipts, quizzes)

View their own analytics (visits, completions, rewards claimed)

2. Super Admin Dashboard (Your View)
2.1 Super Admin – Main Sections

A. Overview / Home

KPI tiles:

Total businesses

Total users

Active missions today

Rewards claimed this week

Quick alerts:

New businesses pending approval

Flagged content (photos/receipts)

System health (API status)

B. Business Management

Table of all businesses:

Name, status (Active/Pending/Suspended), plan, created date, owner email

Actions:

Approve / Reject / Suspend business

Impersonate business admin (log in as them to help troubleshoot)

Edit business info (name, logo, description, category)

Button: “Create New Business” (for manual onboarding)

C. Listings & Missions (Global View)

View all missions across all businesses

Filters:

By business

By location

By status (active, draft, ended)

Actions:

Temporarily disable problematic missions

See mission details & engagement numbers

D. Global Rewards Catalog

Define global reward types:

Points, vouchers, badges, coupons

Templates businesses can reuse:

“Spend $X, earn Y points”

“Visit 3 locations, unlock badge”

Configure:

Expiry rules

Eligible businesses

Global campaigns (e.g., “National Cycling Week”)

E. User & Account Management

View all users:

Customers (participants)

Business admins

Actions:

Reset password / force logout

Block/Unblock user

View user history (missions, submissions, rewards)

F. Moderation

Queue of:

Reported photos

Suspicious receipts

Abusive quiz content

Actions:

Approve / Reject / Flag

Ban user if necessary

G. Analytics & Reports

Platform-wide charts:

Active users over time

Top performing businesses

Most popular missions

Reward redemption rates

Export CSV / PDF for reporting

3. Business Admin Dashboard (For Each Business)
3.1 Business Admin – Main Sections

A. Business Home

KPI tiles:

Total visits/missions started

Completed missions

Rewards claimed

Active missions

Quick actions:

“Create new mission”

“Create new reward”

“View pending submissions”

B. Listings / Locations

Manage:

Store locations / outlets

Each with address, map, opening hours, QR code

Actions:

Add / edit / deactivate locations

Print/display location QR codes

C. Missions / Campaigns
Each business can create “missions” at their locations:

Mission fields:

Title, description

Location(s)

Type of tasks:

Photo submission

Receipt upload

Quiz completion

QR check-in

Validity (start/end dates)

Reward linked

Mission list:

Name, status (Draft/Active/Ended), completions, conversion rate

Actions:

Clone mission

Pause / end mission

Edit mission content

D. Rewards

Create & manage:

Points-based rewards (e.g., 100 pts = free drink)

Coupons (percentage or fixed discounts)

Badges / achievements

For each reward:

Title, description

Required points or conditions

Expiry date

Redemption rules (e.g., in-store only)

View:

Redemption history

Top rewards by popularity

E. Submissions & Verification
For missions with photo/receipt/quiz:

Table of submissions:

User name

Mission name

Type (photo/receipt/quiz)

Status (Pending/Approved/Rejected)

Date

Actions:

View photo/receipt

Check quiz answers

Approve / reject

Leave internal notes (“receipt unclear”, etc.)

F. Users / Participants (Business-Scoped)

List of users who interacted with that business:

Name, email, last active, missions completed, rewards claimed

Actions:

View history

Block from future missions (if cheating)

Grant manual rewards (goodwill gesture)

G. Business Analytics

Charts for that business:

Visits over time

Missions started vs completed

Reward redemption rate

Most popular mission

Funnel: View → Start → Submit → Approved → Reward
